Alpha_0002

完成日期：2013/02/06

類型：車子

材料：01_0001*1、03_0001*2、齒輪*2、積木、輪子*4、金屬輪軸*2、9V電池、電池扣。

 

為彌補Alpha_0001耗損高的缺點，本次使用齒輪傳動，確實提高效率，運作速度遠比Alpha_0001快速。初次使用齒輪之經驗可望做為日後製作01_系列齒輪盒的基礎。礙於材料限制，Alpha_0002的結構和Alpha_0001一樣，並不穩定，容易發生運作中途爆裂、零件脫落等狀況。
